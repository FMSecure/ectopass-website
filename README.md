# ectopass-website

This is the code and infrastructure to deploy the website at https://www.ectopass.com

- It is using GitHub Pages with a custom domain name, ectopass.com
- [Hugo](https://gohugo.io/) with the [Bear Blog theme](https://themes.gohugo.io/themes/hugo-bearblog/) is used to render Markdown to HTML
- [`mdformat`](https://github.com/hukkin/mdformat) is used to format markdown files
